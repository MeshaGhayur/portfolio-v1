Made with Love by Minal Website

I designed and developed a multi-page responsive website for a local bakery business called Made with Love by Minal. Using bootstrap as a basic framework, I built the site using HTML5, SASS and JavaScript. The website is cross-browser compatible, SEO friendly and complies with WCAG AA accessibility standards. The website includes custom fonts, JavaScript carousels and a full-page mobile navigation.
